Build - I have left the .exe in the directory. however if you want to build the program again
just simply you CMake to build the binaries, & take the .exe, glew32.dll, OpenAL32.dll & SDL2.dll files 
and leave them inside the main myengine directory. it should work from there.

W - Up
S - Down
A - Left
D - Right
Space - Play Sound

the object of the "Game" is to collide with all of the cats.

the engine has 3D Rendering & Texturing, Collision, Audio and Input Commands.
